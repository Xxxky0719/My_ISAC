# PIDV 图片搜索复制工具 (v1.1.0)

## 工具简介
这是一个专为自动化处理图片搜寻而设计的 Windows 桌面工具。它能够根据指定的 PID 号和步骤编号（Step），在目标文件夹中快速检索符合命名规则的 JPG 图片，并将其统一复制到指定目录。

## 核心功能
- **三种操作模式**：
  - **单次处理模式**：手动输入单个 PID 和步数进行快速搜索。
  - **CSV 导入模式**：通过上传 CSV 文件进行成千上万条数据的大规模自动化处理。
  - **多行输入模式**：直接在界面粘贴多行数据进行灵活处理。
- **智能命名解析**：
  - 支持识别包含 `PIDV_` 前缀的复杂文件名。
  - 自动解析 `U` (Upper) 和 `L` (Lower) 标识。
  - 支持步骤后缀 `N` (Normal) 和 `T` (Special Test) 的识别与匹配。
- **执行报告与统计**：
  - **自动去重**：复制时若目标文件夹存在同名文件，将自动重命名（如 `_1`, `_2`），防止覆盖。
  - **遗漏统计**：任务完成后，自动生成 `MissingPids_时间戳.txt` 报告，列出所有未找到对应图片的 PID。
- **配置记忆**：自动保存上次使用的搜索路径、目标路径及 CSV 文件路径。

## 命名匹配逻辑
本工具采用精确的正则表达式匹配文件名，标准匹配模式如下：
`...[PIDV_]{PID}[L/U][Step][N/T]...`

**示例解析：**
文件名：`ZH1_MUL201_PIDV_100U01N.jpg`
- 识别出的 PID：`100`
- 识别出的方位：`U`
- 识别出的步骤：`1N`

**匹配细节：**
1. **PID 提取**：优先提取文件名中 `PIDV_` 之后的内容作为 PID；若无此标记，则尝试提取 `L/U` 之前的内容。
2. **步数标准化**：自动忽略输入或文件名中的前导零（例如步数 `01` 会自动匹配 `1`）。
3. **后缀兼容**：如果 CSV 中步数写为 `1`，默认匹配文件名中的 `1N`；如果 CSV 中指定为 `1T`，则精确匹配 `1T`。

## 使用说明
### 1. 准备工作
- 确保图片文件以 `.jpg` 或 `.jpeg` 结尾。
- 确认搜索文件夹中图片命名符合上述逻辑。

### 2. 批量处理 (CSV 模式)
- **准备 CSV**：CSV 文件第一列为 PID，第二列为步数（可选）。
  - *示例内容：*
    ```csv
    PID,Step
    100,01
    101,05T
    102,
    ```
- **执行步骤**：
  1. 选择“CSV 导入模式”。
  2. 点击“浏览”选择 CSV 文件。
  3. 选择“搜索文件夹”和“目标文件夹”。
  4. 点击“执行 CSV 导入”。

### 3. 查看结果
- 界面下方显示最近 5 条执行日志。
- **缺失报告**：如果存在未找到的文件，程序会弹窗提示，并在程序运行目录下的 `Logs` 文件夹中生成详细清单。

## 技术规格
- **运行环境**：Windows 10/11 x64
- **技术栈**：.NET 10.0 Windows Forms
- **发布形式**：单文件自包含 (Single File)，无需额外安装 .NET 运行时。

## 常见问题
- **为什么提示“未找到匹配”？**
  - 请检查文件名是否包含 `L` 或 `U` 字符，且步骤后是否有 `N` 或 `T`。
  - 请确保搜索文件夹路径正确，且文件后缀为 `.jpg`。
- **CSV 读取错误？**
  - 请确保 CSV 文件以 UTF-8 编码保存，或直接使用 Excel 导出的标准 CSV。
